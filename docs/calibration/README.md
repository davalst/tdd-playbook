# Calibration scoreboard

Public record of the Playbook's own planted-defect calibration (see `calibration/` for the
harness). Each scheduled run appends to `history.md`: per-guard and per-agent catch rate,
corpus size and newest-plant date, and any BLOCKING failures with their fixes.

The claim this backs: *our verifiers are tested on a schedule against plants — including
plants they have never seen — and here are the numbers.* A gap in the history is itself a
finding (the schedule is part of the product — §13).
